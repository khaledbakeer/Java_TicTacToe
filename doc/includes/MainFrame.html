      • loadNewGame

        private void loadNewGame()
        Load New Game
        1. Set the Counter to 0 setN(Integer)
        2. Set Player Names setPlayerNames(String, String)
        3. Init. new 2d Object data (empty one). The data object is used to store the movements on it to get it later back as loadData loadData object
        4. As new player the X start always firstly
        5. ReEnable all buttons
        6. Clear the movements list
        7. Remove all button texts and set it to ""
